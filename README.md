# 失效链接检测器

YOURLS 插件。自动探测短链对应的长链接是否失效（返回 4xx/5xx 或无法连接），后台可视化列表展示失效链接，支持一键删除。

---

## 功能特点

- **批量 HEAD 探测**：使用 `get_headers()` 发起 HEAD 请求，追踪 301/302 跳转链，获取最终状态码
- **智能失效判断**：2xx 和 3xx 视为有效，4xx 和 5xx 视为失效，连接超时/失败也视为失效
- **分批 AJAX 检测**：避免一次性检测大量链接导致 PHP 超时，支持配置每批数量
- **实时进度条**：前端可视化展示检测进度，百分比 + 已处理/总数
- **一键删除**：失效链接列表中每条记录均可单独删除，操作即时生效
- **SSL 灵活控制**：支持开关 SSL 证书校验，适配公网域名和私有 OSS / 内网环境

---

## 安装方法

1. 在 YOURLS 安装目录下创建插件文件夹：
   ```
   user/plugins/link-checker/
   ```

2. 将 `plugin.php` 放入上述文件夹：
   ```
   user/plugins/link-checker/plugin.php
   ```

3. 登录 YOURLS 后台 → **管理插件** → 找到 **失效链接检测器** → 点击 **激活**

---

## 使用说明

### 后台配置

激活后，在后台左侧 **插件** 菜单下会出现 **失效链接检测** 子页面。

#### 检测超时（秒）

单个链接的最大等待时间。超时则视为失效。建议 **5~15 秒**。

#### 每批数量

每次 AJAX 请求检测的链接数量。数量过大可能导致 PHP 执行超时，建议 **30~100**。

#### SSL 证书校验

- **开启**：验证远程服务器 SSL 证书（公网域名建议开启）
- **关闭**：允许自签名证书（私有 OSS / 内网域名建议关闭）

### 开始检测

1. 点击 **点击开始检测全部链接** 按钮
2. 进度条实时显示检测进度
3. 检测完成后，失效链接以表格形式列出
4. 每条失效链接右侧有 **删除** 按钮，点击即可删除

---

## 注意事项

- **HEAD 请求**：检测使用 HEAD 方法而非 GET，不下载响应体，节省带宽
- **跳转追踪**：`max_redirects` 设置为 5，可覆盖大多数 CDN / OSS 回源场景
- **PHP 版本要求**：`get_headers()` 的第三个参数需要 PHP 7.1+，请确保服务器环境满足要求
- **超时处理**：检测期间请勿关闭页面，分批 AJAX 会自动完成全部检测
